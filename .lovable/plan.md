## Структура схемы

Центральный узел — большой скруглённый прямоугольник с жирной чёрной обводкой:
- крупный заголовок «Вся правда»
- подзаголовок «в одном месте» обычным весом

Вокруг центра — 6 узлов в светло-серых скруглённых рамках (тонкая граница, без заливки/цвета, без красных подписей):
- сверху по центру: Стратегические KPI
- снизу по центру: Решения
- слева сверху: Реальные данные
- слева снизу: Правила расчёта
- справа сверху: Отчёты
- справа снизу: ИИ-агенты

Соединения — прямые тонкие серые линии от края центрального блока к краю каждого периферийного узла (вертикальные для верх/низ, горизонтальные для левых/правых).

## Десктоп (≥ md)

Сетка 3 колонки × 3 ряда, более широкая (как первый референс):
```
[ . ] [ KPI ] [ . ]
[ Реал.данные ] [ ЦЕНТР ] [ Отчёты ]
[ Правила расч. ] [ ЦЕНТР ] [ ИИ-агенты ]
[ . ] [ Решения ] [ . ]
```
Центральный блок занимает 2 ряда по вертикали (строки 2–3), периферийные узлы слева/справа — по одному на ряд. KPI и Решения — одиночные узлы по центру сверху/снизу.

Пропорции близкие к референсу: ширина ~ 960–1100px, высота ~ 480–540px (соотношение ≈ 2:1).

## Мобильный (< md)

Более квадратная композиция (как второй референс): та же 3×4 раскладка, но уже и выше:
- ширина под контейнер
- центральный блок уже, текст «Реальные / данные» и «Правила / расчёта» переносится в 2 строки
- размеры шрифтов уменьшены, отступы компактнее
- соотношение ближе к 1.4:1

## Линии-соединители

Реализация через абсолютно позиционированные `div` с фоновым цветом (тонкая серая линия 1–2px), привязанные к сетке, либо через один SVG-оверлей с прямыми `<line>` от центра грани центрального блока к центру грани каждого периферийного узла. Предпочту SVG-оверлей внутри `relative`-контейнера — стабильнее при ресайзе и совпадёт с референсом точь-в-точь.

## Стилизация узлов

- border: 1px solid (нейтральный серый, ~ `border-black/15`)
- border-radius: ~ 14px
- фон: прозрачный/белый, без теней, без неоновых эффектов
- текст: чёрный, bold для основных подписей
- центральный блок: border 2px solid почти-чёрный, radius ~ 20px, крупнее по типографике
